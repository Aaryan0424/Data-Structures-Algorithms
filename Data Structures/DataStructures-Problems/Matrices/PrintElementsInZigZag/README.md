# Print Elements In ZigZag

Given an array of dimensions ***mxn***.
We need to print its elements in zigzag order.

### Example

For the given array:
```
    arr(3x3):   1   2   3
                4   5   6
                7   8   9
```
Our spiral output would be:
```
    Output: 1 2 3 6 5 4 7 8 9
```
