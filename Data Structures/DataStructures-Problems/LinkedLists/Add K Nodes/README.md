# Add K Nodes
Given an integer LL. Modify the LL by adding continuous k nodes and inserting a new node after every k nodes indicating their sum. See the sample input for better explanation.

### Sample Input
```
    1 2 3 4 5 6 7 -1
    3
```
### Sample Output
```
    1 2 3 6 4 5 6 15 7 7
```
